Architecture

一条面向安全运营的产品链路，从采集到留痕，前后端都已经成型。

这个项目的价值不只在于“做了什么监控”，还在于把多种监测结果统一汇聚到一个可访问、可管理、可展示的前端入口里， 形成完整的产品化表达。

Product Spine

Go 负责统一服务编排，Python 负责监测与分析，Web 页面负责展示与操作。

SecFlow 通过 Go 后端集中提供静态页面、WebSocket、任务管理与聚合 API， 再由 Python 分析脚本承担抓包、行为分析、局域网画像、网卡流量采集等监测任务， 最终以统一的产品视图输出给值守人员、演示对象或汇报场景。

统一后端 Go 提供 API、WebSocket、任务恢复与多模块聚合
分析引擎 Python 负责抓包、流量采集、行为分析与局域网监测
证据留存 SQLite 记录报文事件、会话、资产画像与日志
交付方式 支持本地运行、静态页面展示与统一控制台交付

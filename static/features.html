AI 驱动的智能网络安全分析平台

🤖 AI 威胁检测引擎

集成 Ollama 和 Llama.cpp 大语言模型，对每个数据包进行深度语义分析。AI 能够识别传统规则难以发现的复杂攻击模式，包括 DDoS、端口扫描、SQL 注入、异常流量等。

  • 支持 DeepSeek-R1、Llama 等多种模型
  • 实时威胁评分和分类
  • 自动生成威胁分析报告
  • 可配置本地或远程 AI 服务

🛡️ 自动防火墙联动

检测到威胁后，系统自动调用操作系统防火墙 API 封锁恶意 IP。支持 Windows Defender Firewall 和 Linux iptables，内置白名单机制防止误封锁。

  • Windows: netsh advfirewall 自动规则
  • Linux: iptables DROP 规则
  • 网关/DNS/内网 IP 白名单保护
  • 实时显示防火墙动作状态

📊 上网行为智能分析

通过监控 DNS 查询流量，AI 聚合分析用户访问的网站域名，推测用户行为（观看视频、编程、购物、搜索等），生成可视化的上网足迹时间线。

  • DNS 流量聚合（15秒窗口）
  • AI 识别应用名称和行为
  • 风险等级评估（安全/可疑/危险）
  • 实时时间线展示

14:35 🛍️ 淘宝

⚡ 实时 WebSocket 架构

Go 后端提供高性能 WebSocket 服务，Python 脚本通过标准输出流式传输 JSON 数据，前端实时接收并渲染。支持双向通信，可随时启动/停止捕获任务。

  • 毫秒级数据推送延迟
  • 支持多客户端并发连接
  • 优雅的进程管理和清理
  • 跨平台进程控制（Windows/Linux）

🎯 灵活的流量过滤

支持基于端口的 BPF 过滤器，可精准捕获 HTTP(80)、HTTPS(443) 或自定义端口流量。减少无关数据包干扰，提升 AI 分析效率。

  • 预设 HTTP/HTTPS 快捷过滤
  • 自定义端口号输入
  • 支持 TCP/UDP 协议筛选
  • 可配置抓包数量限制

过滤配置

Port Filter: 443 (HTTPS)
Packet Count: 50
Interface: Wi-Fi
BPF: "port 443"

🔒 安全与隐私保护

所有数据包分析在本地完成，不上传到云端。AI 模型可部署在本地（Ollama）或私有服务器（Llama.cpp），确保敏感数据不泄露。

  • 本地数据处理，无云端传输
  • 支持本地 AI 模型部署
  • 管理员权限检测提示
  • 白名单机制防止误操作
